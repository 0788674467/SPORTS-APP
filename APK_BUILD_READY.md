## 📱 Build Your APK (3 Easy Steps)

### Step 1: Go to GitHub Actions
Open this link in your browser:
```
https://github.com/0788674467/SPORTS-APP/actions
```

### Step 2: Run the Workflow
1. Click **"Build Android APK"** in the left sidebar
2. Click the **"Run workflow"** button (top right, green button)
3. Click **"Run workflow"** again in the dropdown
4. Wait 5-10 minutes ⏱️

### Step 3: Download APK
1. Click on the completed workflow run (green ✅ checkmark)
2. Scroll down to **"Artifacts"** section
3. Click **"app-release-apk"** to download
4. Extract the ZIP file
5. You'll find **`app-release.apk`** inside
